## 背景与需求分析（调研结论）

"看大盘"的公认框架 = **宏观 → 技术 → 资金 → 情绪 → 板块 五步法**。对照本项目数据家底：

| 维度 | 要看什么 | 数据现状 |
|---|---|---|
| 技术·指数走势 | 走势图 + 均线(站上/跌破20/60、多头排列) | `index_daily` 仅收盘价（无 OHLC/量）→ **收盘线+MA20/60 可做**；真K线需补 OHLC（本期不做） |
| 技术·量能 | 两市成交额趋势(放量/缩量) | `daily_bar` 按日 `sum(amount)` ✅ |
| 资金·大盘主力 | 主力净流入趋势 | 已有 `market-fund` ✅ |
| 资金·板块 | 板块资金排行/热力 | 已有 `sector-fund` ✅（待盘中灌数） |
| 情绪 | 涨跌家数/涨停跌停/温度 | 已有 `sentiment` ✅ |
| 板块 | 领涨/轮动/异常 | 已有 `sectors`/`abnormal` ✅ |
| 资金·北向、情绪·连板、宏观·政策 | — | ❌ 无源（本期不做） |

**病因**：现 /macro 五维料都有一点，但（1）全是数字**没有图**（尤其指数）；（2）没有**一句话研判结论**（其实 `market.regime()` 已能给 进攻/均衡/防守+建议仓位）。

## 关键决策（已确认）

- 指数走势档位 = **C：先收盘线 + MA20/60，真K线(OHLC)以后补**。
- 驾驶舱按**五步法**组织，**每块配图**，**顶部放一句话研判**（regime）。
- 北向/连板/政策不做（无源）。

## 第 1 节 · 后端（少量新增）

`aquant/macro.py` 新增（只读库、无联网）：
- `regime() -> dict`：直接返回 `market.regime()`（state/score/suggested_position/note/breadth/index）。
- `index_series(code="sh000300", n=120) -> dict`：读 `index_daily` 该指数最近 n 日收盘，算 MA20/MA60 → `{code, points:[{date, close, ma20, ma60}]}`（ma 不足窗口为 None）。
- `amount_trend(days=20) -> dict`：`daily_bar` 按 date `sum(amount)` 最近 days → `{series:[{date, amount}]}`（amount 单位亿元，/1e8）。

端点（挂在既有 macro/cockpit 路由）：
- `GET /api/cockpit/regime` → 研判
- `GET /api/cockpit/index-series?code=sh000300&n=120` → 指数走势+均线
- `GET /api/cockpit/amount-trend?days=20` → 两市量能

复用不变：`/sentiment`、`/market-fund`、`/sector-fund`、`/abnormal`、`/sectors`。

## 第 2 节 · 前端（/macro 重做）

`pages/Cockpit.tsx`（/macro）重构为"看大盘"五步法，全部配图、顶部研判：

- **顶部·今日研判**：`regime` → 进攻/均衡/防守 徽章 + 建议仓位 + note 一句话。
- **技术**：
  - 指数走势图（`index-series` → 收盘线 + MA20/MA60 折线，图例，标"站上/跌破 MA20/60"）
  - 量能柱（`amount-trend` → 两市成交额趋势，放量红/缩量灰或按环比着色）
- **资金**：大盘主力净流入趋势（`market-fund` 已有）+ 板块资金热力（`sector-fund` 已有）
- **情绪**：涨跌家数对比条（`sentiment` → 上涨/下跌 横向条）+ 涨停/跌停 + 温度徽章
- **板块**：领涨板块（`sectors`）+ 异常资金（`abnormal`）

新增图表 option 纯函数：`buildIndexTrendOption(points)`（多折线：close/ma20/ma60）、`buildAmountBarOption(series)`（量能柱）、`buildBreadthBarOption(sentiment)`（涨跌家数对比条）。沿用深色原子 + EChart 薄壳。

## 非目标（YAGNI）

- 不做指数真K线(OHLC)——本期 C 档只做收盘线+MA；OHLC 入库留后续。
- 不做北向、连板高度/炸板率、政策新闻（无源/超范围）。
- 不新增指数（默认沪深300；上证/创业板若 index_daily 有则顺带，无则跳过）。
- 不动看板/持仓/量化功能。
