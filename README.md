# A股量化投研系统（aquant）

本地个人量化投研 + 看盘工具。最终目标：**帮你回答「选什么股、什么时候买卖」**。

## 设计

| 层 | 模块 | 职责 |
|---|---|---|
| 数据 | `aquant/data` | AkShare 取数 → DuckDB 本地仓库，增量更新 |
| 因子 | `aquant/factors` | 技术/资金/基本面因子计算 + 因子注册表 |
| 选股 | `aquant/select` | 多因子打分排序 → 候选股票池 |
| 信号 | `aquant/signal` | 择时买卖点信号 |
| 回测 | `aquant/backtest` | 向量化回测 + 绩效（年化/夏普/回撤/胜率）+ 归因 |
| 看盘 | `aquant/dashboard` | Streamlit：资金流热力、板块轮动、龙虎榜、信号面板 |

技术选型：Python 3.11 · AkShare（免费数据源）· DuckDB+Parquet（本地列式存储）· Plotly/Streamlit（可视化）· 自研轻量向量化回测。

## 快速开始

```bash
pip install -r requirements.txt

python -m aquant.cli basic                  # 更新股票列表
python -m aquant.cli daily 600519 000001    # 拉/增量更新日线
python -m aquant.cli summary                # 仓库概览
python -m aquant.cli kline 600519           # 导出K线图
python -m aquant.cli flow                   # 个股资金流当日快照
python -m aquant.cli sectors                # 行业板块当日快照
```

数据落在 `data_store/`（已 gitignore）：`market.duckdb` 主库 + `parquet/`。

## 研究发现（持续更新）

基于 5207 只全 A、2018–2026 数据的实证结论：

1. **5 日尺度上动量是反向的**（IC 检验）：mom/trend/换手/RSI/MACD 的 IR 全为
   −0.25~−0.42；唯一稳定正向的是 **低波动**(IR +0.47) 与 **不追高/低乖离**(IR +0.37)。
   故 `scorer.DEFAULT_WEIGHTS` 已采用数据驱动的 `IC_WEIGHTS`（低波+反转+不追高）。
2. **换手成本是高频选股的死因**（全市场 5122 只复验稳健）：IC 加权 Top50，
   周度换仓 −45%、月度 +8%、季度换仓 **+93%**（年化 8.5%/夏普 0.32/回撤 −40%）。
   → 正确用法：低波选股 + **季度持有**，不要天天倒腾。
3. 真正的 alpha 主要来自**低波动异象**（慢变量、低换手可捕获）；快速反转 alpha 真实
   但被成本吃光。
4. **对可实操宽基指数有真实超额**（同期 2018–2026，清洁域 4473 只）：季度 IC 策略
   +90.9%（年化 8.3%/夏普 0.31/回撤 −39%）vs 沪深300 +19.7% / 中证500 +32.0%。
   （唯一跑输的「等权全市场」不可实操，是小盘 beta 机器，非合理对标。）
6. **选股域必须剔除 ST/次新/低流动性**：ST 股 5% 涨跌停使其波动人为偏低，会被低波因子
   误选（实盘会买到退市风险股）。剔除 649 只后回测几乎不变（+96%→+91%）——**超额不是
   ST 假象**，但实盘过滤不可省。见 `research.universe()`。

> 结论曾两次反复：先因 400 只主板样本高估（+112% 假超额）、再因等权基准低估（误判跑输），
> 全市场 + 沪深300/中证500 公平基准复验后才定稿——样本偏差与基准选择的典型教训。
> 仍未处理退市幸存者偏差，实盘前需进一步打折。

5. **朴素指数趋势 regime 择时不提升夏普**：沪深300>MA200 把回撤从 −40% 降到 −32%，
   但收益 +96%→+61%、夏普基本不变（0.33→0.32）——只是缩放 beta 暴露、未创造 alpha。
   MA120 更差（过敏感被打脸）。想压回撤可选 MA200（半仓代价），别指望提升风险调整后收益。
7. **组合内风险平价(反波动率)配权几乎无增量**（清洁域 4446 只，季度换仓 Top50）：
   按近 20 日波动率倒数配权 vs 等权——夏普 0.413→0.421、年化波动 26.4%→25.7%、
   回撤 −38.0%→−37.3%，收益反略降（年化 10.9%→10.8%），全是 noise 级。
   **根因**：候选池已在选股阶段被低波因子筛过，Top50 内部「一片低波」无区分度，
   低波 alpha 在**选股**环节已捕获，**配权**环节重复做无增量。代码已加
   `backtest_topn(weighting="inv_vol")` 作可选项，但不设默认。教训同 regime 择时——
   别为听起来高级的优化买单。见 `scripts/weighting_compare.py`。
